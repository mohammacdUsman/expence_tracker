# 💰 Expensify — Split Expenses with Ease

A modern, browser-based expense tracking application designed to simplify the process of splitting bills and managing shared expenses among friends, roommates, and colleagues.

---

## 🚀 Features

✨ **Smart Expense Splitting**
- Effortlessly add and manage shared expenses
- Automatically calculate individual contributions
- Support for equal and custom splits

👥 **Member Management**
- Add multiple participants to expense groups
- Store member profiles with avatars
- Edit and remove members anytime

💳 **Payment Tracking**
- Track who owes whom with real-time calculations
- Monitor payment status at a glance
- Visualize settlement workflows

📊 **Professional Reports**
- Generate detailed expense summaries
- Export data to PDF for records and sharing
- Beautiful UI with professional design

🎨 **Modern User Interface**
- Clean, intuitive design with responsive layout
- Dark mode with elegant purple & gold theme
- Mobile-friendly experience
- Smooth animations and transitions

💾 **Browser-Based**
- No installation required
- Works directly in your web browser
- Fast and lightweight performance

## 🛠️ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Quick Start

1. **Open the Application**
   ```
   Simply open index.html in your web browser
   ```
   - Double-click the file, or
   - Drag and drop into your browser, or
   - Use "File → Open" in your browser menu

2. **Create Your First Expense Group**
   - Add members to your expense group
   - Enter member names and add profiles
   - Start adding expenses

3. **Add Expenses**
   - Click "Add Expense" button
   - Enter amount, description, and who paid
   - Select participants and their shares
   - Confirm to save

4. **Track Payments**
   - View real-time settlement summary
   - See who owes whom
   - Export reports when needed

### Local Development

To serve the application locally:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (with http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
